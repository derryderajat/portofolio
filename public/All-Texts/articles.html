    There is a new way to use the group by operation in JavaScript!

    (It's very easy)

    In the past, grouping array elements by a property was typically done by using the `Array.reduce` method with questionable syntax.

    Recently, they've added support for the Object.groupBy method, which makes grouping elements really easy.

    The first argument is the array, the second is the callback function that returns the keys in the grouped object 👇

    const obj = Object.groupBy(arr, (el) => el.type);
